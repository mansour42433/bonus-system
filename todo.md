### جدول الفواتير (Invoices)
- [ ] ربط الفاتورة بالمندوب (repEmail)
- [ ] تخزين: رقم الفاتورة، العميل، التاريخ، الإجمالي
- [ ] إضافة حقل invoiceStatus (Paid/Unpaid)
- [ ] إضافة حقل paymentDate

### جدول حالة البونص (BonusPayments)
- [ ] إنشاء جدول bonusPayments
- [ ] حقول: invoiceId, repEmail, bonusAmount, bonusPercentage
- [ ] حقل bonusStatus (paid/unpaid)
- [ ] حقل bonusPaymentDate

### جدول المنتجات (Products)
- [ ] التحقق من وجود جدول products
- [ ] حقول: productId, productName, category, price

### جدول تفاصيل الفواتير (InvoiceItems)
- [ ] ربط الفاتورة بالمنتجات
- [ ] حقول: invoiceId, productId, quantity, price, total
- [ ] حقل category (من المنتج)

### جدول المرتجعات (CreditNotes)
- [ ] التحقق من وجود جدول creditNotes
- [ ] حقول: creditNoteId, invoiceId, productId, quantity, amount
