# نظام العمولات والبونص - المهام

## المطلوب الأساسي: إدارة تسليم البونص
- [x] checkbox بجانب كل فاتورة في جدول البونص المستحق لتحديدها
- [x] زر "تم التسليم" لحفظ الفواتير المحددة كمسلمة في قاعدة البيانات تلقائياً
- [x] عزل الفواتير المسلمة في تبويب منفصل "بونص مسلم"
- [x] الفواتير غير المسلمة تظهر في تبويب "بونص غير مسلم"
- [x] حفظ تلقائي في قاعدة البيانات عند التسليم (invoiceId + repEmail + bonusAmount + تاريخ التسليم)
- [x] تصدير Excel متقدم بـ 5 أوراق عمل (ملخص، غير مسلم، مسلم، آجلة، منتجات)

## مهام سابقة مكتملة
- [x] نظام حساب البونص الأساسي (1% و 2%)
- [x] جلب الفواتير من Qoyod حسب تاريخ الدفع
- [x] استبعاد المرتجعات من حساب البونص
- [x] إعدادات المنتجات (سعر التميز، تفعيل/إيقاف البونص)
- [x] إعدادات المندوبين (الأسماء المستعارة)
- [x] إصلاح مشكلة رول تاريخ وباركود 60*40
